# 三分钟守住防线

一个自动攻击、左右走位、防守 Boss 的小游戏。支持两种玩法：

- 单机：直接打开 `index.html`
- 局域网联机：电脑启动 `server.js`，好友用浏览器访问电脑的局域网地址

## 单机玩法

直接双击打开：

```text
index.html
```

这种方式不需要服务器，适合自己测试单机版。

## 局域网联机玩法

联机需要一台电脑当服务器。所有玩家的手机或电脑必须和服务器电脑在同一个局域网，比如连同一个 Wi-Fi。

### 1. 安装 Node.js

电脑需要先安装 Node.js。

打开命令行后输入：

```bash
node -v
```

如果能看到版本号，说明已经安装好了。

### 2. 启动服务器

在游戏文件夹打开命令行：

```text
E:\Documents\游戏
```

然后运行：

```bash
node server.js
```

成功后会显示类似：

```text
局域网联机服务器已启动: http://localhost:8787
手机访问: http://192.168.x.x:8787
```

### 3. 自己打开游戏

服务器电脑可以打开：

```text
http://localhost:8787
```

### 4. 好友加入

好友在手机或电脑浏览器里输入服务器显示的地址，例如：

```text
http://192.168.200.1:8787
```

注意：好友不能访问 `localhost`，`localhost` 只代表他们自己的设备。好友必须访问服务器电脑的局域网 IP。

### 5. 开始游戏

每个玩家进入页面后选择角色。

任意一名玩家点击“开始游戏”，所有在线玩家会进入同一局。第一版最多支持 4 人合作。

## 如何查看电脑局域网 IP

如果启动服务器时没有看到正确地址，可以在 Windows 命令行输入：

```bash
ipconfig
```

找到类似下面的 IPv4 地址：

```text
IPv4 Address . . . . . . . . . . . : 192.168.x.x
```

好友访问：

```text
http://这个IPv4地址:8787
```

## 常见问题

### 好友打不开页面

先检查：

- 好友和服务器电脑是否连同一个 Wi-Fi
- 服务器电脑是否还开着 `node server.js`
- 地址是否写成了 `http://电脑IPv4地址:8787`
- 不要让好友访问 `http://localhost:8787`

如果仍然打不开，可能是 Windows 防火墙拦截了 Node.js。允许 Node.js 通过专用网络即可。

### 电脑自己能打开，手机打不开

通常是 IP 选错或防火墙问题。

电脑自己用：

```text
http://localhost:8787
```

手机用：

```text
http://电脑IPv4地址:8787
```

### 如何停止服务器

在运行 `node server.js` 的命令行窗口按：

```text
Ctrl + C
```

如果是后台启动的服务器，可以关闭对应的 Node.js 进程。
